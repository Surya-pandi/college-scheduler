# College Staff Scheduler

A comprehensive staff scheduling system for colleges with automatic rescheduling and leave management.

## Features

- **Admin Dashboard**: Manage staff, create timetables, approve leave requests
- **Staff Dashboard**: View schedule, mark attendance, apply for leave
- **Automatic Scheduling**: Automatically find replacements when staff are on leave
- **Leave Management**: Staff can request leave, admins can approve/reject
- **Attendance Tracking**: Track staff presence/absence
